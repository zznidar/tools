PDB Insertion Codes Remover

This tool renumbers your residues if they contain insertion codes.

Please note that it completely messes up with any sheets, helices, disulphide bonds and similar remarks. Therefore, they are removed from the output .pdb altogether.

Select a .pdb file, conversion will happen automatically and download will be initiated.



If you wish to fill in missing gaps, check out the PDB Missing Residues Contig Maker tool.
